Requirements:
Download and Install any local web server such as XAMPP/WAMP.
Download the provided source code zip file.

Installation/Setup:
Open your XAMPP/WAMP's Control Panel and start Apache & MySQL.
Extract the downloaded source code zip file.
Import inventory.slq to localhost database
If you are using XAMPP, copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory.
Browse the inventory in a browser. i.e. http://localhost/inventory/index.php
